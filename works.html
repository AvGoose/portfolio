Liiku
A Fitness Application Prototype Made Using Figma.

Liiku was a joint effort with a fellow student, where we built a fully functional prototype using Figma.

Aesthetics, Use Experience design and functionality were made by me.

Vessa.info
A web application that shows nearby public bathrooms in the Pääkaupunkiseutu area.

Vessa.info was created with a team of other students.

I came up with the actual idea for the application, and built the User Interface and participated in the User Experience design.

Vessa.info was created using React.js, and Material UI.

GoseBeats
GoseBeats is an audiovisual project I work on my spare time.

Gosebeats is an audiovisual project, where I produce beats using Ableton ("instrumentals") and accompany them with graphics designs made using Artrage and GIMP.

I designed a website for this project using SquareSpace, maintaining both a protfolio and a webshop.
